Angular 7+

- https://angular.io/

Pré requisitos:

- NodeJs:

    - sudo apt-get install nodejs npm


Angular

- Instalação:

    - npm install -g @angular/cli

Versões:

$ node -v

$ npm -v

$ ng --version

Start Server:

$ ng serve --open

Angular Cli:

    - Iniciar novo projeto

        - ng new [nome-do-projeto]

    - Criar Componente

        - ng g c [nome-do-componente]
    
    - Criar Service

        - ng g service [nome-do-service]
